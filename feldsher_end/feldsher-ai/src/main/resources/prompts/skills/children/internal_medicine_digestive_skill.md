# 消化内科 Skill (Internal Medicine Digestive Skill)

## 科室概述

消化内科主要处理消化系统疾病，包括食管、胃、肠、肝、胆、胰等器官的疾病。

## 常见疾病

| 疾病 | 典型症状 |
|------|----------|
| 慢性胃炎 | 上腹痛、腹胀、反酸、嗳气、食欲不振 |
| 消化性溃疡 | 周期性、节律性上腹痛，胃溃疡餐后痛，十二指肠溃疡空腹痛 |
| 胃食管反流病 | 反酸、烧心、胸骨后疼痛、咽部异物感 |
| 功能性消化不良 | 上腹胀、早饱、嗳气、食欲不振 |
| 急性胃肠炎 | 腹痛、腹泻、恶心、呕吐、发热 |
| 肠易激综合征 | 腹痛、腹泻或便秘，排便后缓解 |
| 胆囊炎 | 右上腹痛、可放射至右肩、油腻饮食诱发 |
| 肝炎 | 乏力、食欲不振、恶心、黄疸、肝区不适 |

## 问诊模板

### 第一部分：主诉相关（必须询问）

#### 1. 症状特点
- [ ] 这个症状是从什么时候开始的？
- [ ] 症状持续了多久？是阵发性还是持续性？
- [ ] 每天大概发作几次？每次持续多久？
- [ ] 这种不舒服是什么样的感觉？（如隐痛、胀痛、刺痛、绞痛、烧灼样）
- [ ] 疼痛的程度如何？（1-10分，10分最痛）

#### 2. 部位与放射
- [ ] 具体是哪个位置不舒服？（可以用手指指一下）
- [ ] 疼痛会放射到其他地方吗？（如后背、肩部、腰部）
- [ ] 疼痛的位置固定吗？还是会移动？

#### 3. 诱发与缓解因素
- [ ] 什么情况下会加重？（如饭后、空腹、夜间、情绪紧张、特定食物）
- [ ] 什么情况下会缓解？（如空腹、进食、休息、排便、服用药物）
- [ ] 症状发作与饮食有关系吗？
- [ ] 与排便有关系吗？

### 第二部分：伴随症状

#### 1. 胃肠道症状
- [ ] 有没有恶心、呕吐？呕吐物是什么样的？
- [ ] 有没有呕血？呕吐物是咖啡色还是鲜红色？
- [ ] 有没有反酸、烧心？
- [ ] 有没有嗳气、打嗝？
- [ ] 有没有腹胀？什么时候明显？
- [ ] 有没有食欲不振？最近体重有没有变化？

#### 2. 排便情况
- [ ] 大便颜色正常吗？有没有黑便或血便？
- [ ] 大便形状正常吗？有没有稀便、水样便或粘液便？
- [ ] 每天排便几次？还是几天一次？
- [ ] 排便时有没有腹痛？便后腹痛缓解吗？
- [ ] 有没有里急后重（想排便又排不出来）？

#### 3. 全身症状
- [ ] 有没有发热？体温大概多少？
- [ ] 有没有乏力、容易疲劳？
- [ ] 有没有皮肤或眼睛发黄？
- [ ] 有没有皮肤瘙痒？
- [ ] 最近体重有没有明显下降？

### 第三部分：病史相关

#### 1. 既往史
- [ ] 以前有没有过类似的情况？
- [ ] 有没有做过胃镜、肠镜等检查？结果如何？
- [ ] 有没有确诊过什么消化系统疾病？（如胃炎、胃溃疡、胆囊炎等）
- [ ] 有没有其他慢性病？（如高血压、糖尿病、心脏病）
- [ ] 有没有做过腹部手术？

#### 2. 用药史
- [ ] 这次发病后有没有吃过什么药？效果如何？
- [ ] 平时有没有长期服用的药物？
- [ ] 有没有服用阿司匹林、布洛芬等止痛药？
- [ ] 有没有服用抗生素？

#### 3. 过敏史
- [ ] 有没有对什么药物过敏？
- [ ] 有没有对什么食物过敏？

#### 4. 个人史
- [ ] 吸烟吗？每天多少？吸了多少年？
- [ ] 喝酒吗？喝什么酒？大概多少量？
- [ ] 平时饮食习惯如何？（是否规律、是否偏好辛辣/油腻/生冷）
- [ ] 最近有没有外出就餐或吃了特殊食物？
- [ ] 工作或生活压力大吗？有没有焦虑紧张？

#### 5. 家族史
- [ ] 家人有没有类似的消化系统疾病？
- [ ] 有没有消化道肿瘤家族史？
- [ ] 有没有肝病家族史？

## 鉴别诊断思考

### 根据症状鉴别

| 主要症状 | 可能疾病 | 鉴别要点 |
|----------|----------|----------|
| 上腹痛 | 慢性胃炎、胃溃疡、十二指肠溃疡、胆囊炎、胰腺炎 | 疼痛的时间、部位、诱发因素 |
| 反酸烧心 | 胃食管反流病、消化性溃疡 | 与体位、饮食的关系 |
| 腹痛腹泻 | 急性胃肠炎、肠易激综合征、炎症性肠病 | 起病急缓、粪便特点、伴随症状 |
| 腹胀嗳气 | 功能性消化不良、慢性胃炎、胃下垂 | 与饮食的关系、其他伴随症状 |
| 呕血黑便 | 消化性溃疡、食管胃底静脉曲张、胃癌 | 出血量、颜色、既往病史 |

## 红色警报症状

出现以下情况需要立即建议就医：

- [ ] 呕血或黑便（提示消化道出血）
- [ ] 剧烈腹痛持续不缓解
- [ ] 发热伴严重腹痛
- [ ] 黄疸（皮肤眼睛发黄）
- [ ] 明显体重下降
- [ ] 年龄>50岁新发症状
- [ ] 有消化道肿瘤家族史

## 问诊示例对话

```
用户: 我最近胃痛

医生: 
1. 这个胃痛是从什么时候开始的？
2. 是每天都痛，还是偶尔痛？每次痛多久？
3. 疼痛是什么样的感觉？是隐隐作痛、胀痛、还是像火烧一样？
4. 疼痛具体在哪个位置？是胸口下面、肚脐上面，还是其他地方？
5. 疼痛和吃饭有关系吗？比如是饿的时候痛，还是吃饱了以后痛？
6. 除了胃痛，还有没有其他不舒服？比如反酸、烧心、恶心、呕吐？
7. 大便颜色正常吗？有没有变黑？
8. 以前有没有过类似的情况？有没有做过胃镜检查？
9. 最近有没有吃什么特别的东西？有没有喝酒？
10. 平时有没有长期吃什么药？
```

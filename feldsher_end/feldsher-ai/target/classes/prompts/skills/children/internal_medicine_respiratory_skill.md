# 呼吸内科 Skill (Internal Medicine Respiratory Skill)

## 科室概述

呼吸内科主要处理呼吸系统疾病，包括鼻、咽、喉、气管、支气管、肺、胸膜等器官的疾病。

## 常见疾病

| 疾病 | 典型症状 |
|------|----------|
| 急性上呼吸道感染 | 鼻塞、流涕、咽痛、咳嗽、发热 |
| 急性支气管炎 | 咳嗽、咳痰、可有发热 |
| 慢性支气管炎 | 长期咳嗽、咳痰，每年持续3个月以上 |
| 肺炎 | 发热、咳嗽、咳痰、胸痛、呼吸困难 |
| 支气管哮喘 | 发作性喘息、胸闷、咳嗽、呼吸困难 |
| 慢性阻塞性肺疾病(COPD) | 慢性咳嗽、咳痰、进行性呼吸困难 |
| 肺栓塞 | 突发胸痛、呼吸困难、咯血、晕厥 |
| 肺结核 | 咳嗽、咳痰、咯血、低热、盗汗、乏力、消瘦 |
| 肺癌 | 刺激性咳嗽、咯血、胸痛、呼吸困难、体重下降 |
| 胸腔积液 | 呼吸困难、胸痛、咳嗽 |

## 问诊模板

### 第一部分：主诉相关（必须询问）

#### 1. 咳嗽特点
- [ ] 咳嗽是从什么时候开始的？
- [ ] 是干咳还是有痰？
- [ ] 咳嗽什么时候比较严重？（如晨起、夜间、运动后、遇冷空气）
- [ ] 咳嗽的声音是什么样的？（如刺激性干咳、犬吠样、金属音）
- [ ] 咳嗽是阵发性还是持续性？
- [ ] 咳嗽影响睡眠吗？

#### 2. 咳痰特点
- [ ] 有痰吗？痰是什么颜色的？（白色、黄色、绿色、铁锈色、粉红色）
- [ ] 痰的性状？（稀薄、粘稠、泡沫样、脓性）
- [ ] 每天大概咳多少痰？
- [ ] 痰中带血吗？是血丝还是血块？
- [ ] 咳痰和体位有关系吗？（如躺下时痰多）

#### 3. 呼吸困难
- [ ] 有呼吸困难吗？感觉气不够用？
- [ ] 呼吸困难是突然发生的，还是慢慢出现的？
- [ ] 什么情况下会加重？（活动后、平躺时）
- [ ] 呼吸困难的程度？（能上几层楼？）
- [ ] 有没有端坐呼吸（必须坐着才能呼吸）？
- [ ] 有没有夜间阵发性呼吸困难（睡着后憋醒）？

#### 4. 胸痛
- [ ] 有胸痛吗？具体位置在哪里？
- [ ] 胸痛和呼吸有关系吗？（深呼吸或咳嗽时加重）
- [ ] 胸痛是什么样的感觉？（针刺样、闷痛、压榨样）
- [ ] 胸痛持续多久？

#### 5. 喘息
- [ ] 有没有喘息？感觉像拉风箱一样？
- [ ] 发作时有诱因吗？（如接触花粉、尘螨、冷空气）
- [ ] 喘息发作时能听到哮鸣音吗？
- [ ] 以前有过类似发作吗？

### 第二部分：全身症状

#### 1. 发热
- [ ] 有没有发热？感觉身上发烫？
- [ ] 体温大概多少度？有没有测量过？
- [ ] 发热有规律吗？（午后低热、持续高热）
- [ ] 有没有畏寒、寒战？
- [ ] 有没有出汗？特别是夜间盗汗？

#### 2. 其他全身症状
- [ ] 有没有乏力、容易疲劳？
- [ ] 最近体重有没有变化？是减轻还是增加？
- [ ] 有没有食欲不振？
- [ ] 有没有关节痛、肌肉痛？

### 第三部分：病史相关

#### 1. 既往史
- [ ] 以前有没有过类似的情况？
- [ ] 有没有慢性呼吸系统疾病？（如哮喘、慢阻肺、肺结核）
- [ ] 有没有做过胸部CT、胸片、肺功能等检查？
- [ ] 有没有心脏病、高血压、糖尿病等其他慢性病？
- [ ] 有没有过敏性鼻炎、荨麻疹等过敏性疾病？

#### 2. 用药史
- [ ] 这次发病后有没有用过什么药？效果如何？
- [ ] 平时有没有长期服用的药物？
- [ ] 有没有用降压药？（如ACEI类可能引起干咳）
- [ ] 有没有用抗生素？用了几天？

#### 3. 过敏史
- [ ] 有没有对什么药物过敏？
- [ ] 有没有对什么食物过敏？
- [ ] 有没有季节性过敏？（如花粉症）

#### 4. 个人史
- [ ] 吸烟吗？每天多少支？吸了多少年？
- [ ] 有没有戒烟？戒了多久？
- [ ] 喝酒吗？
- [ ] 平时工作或生活环境怎么样？有没有接触粉尘、烟雾、化学物质？
- [ ] 最近有没有去过人员密集的地方？有没有接触过发热或咳嗽的病人？
- [ ] 有没有养宠物？（猫、狗、鸟等）

#### 5. 家族史
- [ ] 家人有没有类似的呼吸系统疾病？
- [ ] 有没有哮喘家族史？
- [ ] 有没有肺部肿瘤家族史？
- [ ] 有没有过敏性疾病家族史？

## 鉴别诊断思考

### 根据症状鉴别

| 主要症状 | 可能疾病 | 鉴别要点 |
|----------|----------|----------|
| 咳嗽咳痰 | 急性支气管炎、肺炎、慢性支气管炎、支气管扩张 | 起病急缓、痰的性状、伴随症状 |
| 呼吸困难 | 哮喘、COPD、肺炎、肺栓塞、心衰 | 发作特点、伴随症状、既往病史 |
| 咯血 | 肺结核、肺癌、支气管扩张、肺炎 | 咯血量、伴随症状、年龄、吸烟史 |
| 胸痛 | 肺炎、胸膜炎、肺栓塞、气胸、心绞痛 | 与呼吸的关系、部位、性质、伴随症状 |
| 发热伴咳嗽 | 肺炎、急性支气管炎、肺结核、肺脓肿 | 热型、痰的性状、全身中毒症状 |

## 红色警报症状

出现以下情况需要立即建议就医：

- [ ] 严重呼吸困难、端坐呼吸、口唇发绀
- [ ] 胸痛持续不缓解，特别是伴有呼吸困难
- [ ] 大咯血（一次咯血量>100ml或24h>500ml）
- [ ] 意识模糊、烦躁不安
- [ ] 高热不退（>39℃）
- [ ] 呼吸困难进行性加重
- [ ] 有肺栓塞高危因素（长期卧床、手术、肿瘤、妊娠）

## 问诊示例对话

```
用户: 我咳嗽好几天了，还有点发烧

医生:
1. 咳嗽是从什么时候开始的？是突然开始的，还是慢慢出现的？
2. 咳嗽有痰吗？痰是什么颜色的？黄色、白色还是其他颜色？
3. 除了咳嗽，有没有呼吸困难？感觉气不够用？
4. 有没有胸痛？特别是咳嗽或深呼吸的时候？
5. 发烧大概多少度？有没有畏寒、寒战？
6. 有没有流鼻涕、嗓子痛？
7. 最近有没有接触过发热或咳嗽的病人？
8. 以前有没有过类似的情况？有没有哮喘、支气管炎等病史？
9. 吸烟吗？每天吸多少？
10. 这次有没有吃过什么药？效果怎么样？
```

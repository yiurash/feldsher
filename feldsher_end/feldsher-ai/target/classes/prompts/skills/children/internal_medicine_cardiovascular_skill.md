# 心血管内科 Skill (Internal Medicine Cardiovascular Skill)

## 科室概述

心血管内科主要处理心脏和血管系统的疾病，包括冠心病、高血压、心律失常、心力衰竭等。

## 常见疾病

| 疾病 | 典型症状 |
|------|----------|
| 冠心病 | 胸痛、胸闷、活动后加重、休息后缓解 |
| 高血压 | 头痛、头晕、头胀、颈项板紧、疲劳、心悸 |
| 心律失常 | 心悸、心慌、心跳停顿感、胸闷、头晕 |
| 心力衰竭 | 呼吸困难、乏力、水肿、咳嗽、咳痰 |
| 心肌梗死 | 持续剧烈胸痛、胸闷、濒死感、大汗、恶心呕吐 |
| 心瓣膜病 | 活动后呼吸困难、心悸、乏力、水肿 |
| 心肌病 | 呼吸困难、乏力、水肿、心悸 |
| 心包炎 | 胸痛（深呼吸或咳嗽时加重）、呼吸困难、发热 |

## 问诊模板

### 第一部分：主诉相关（必须询问）

#### 1. 胸痛/胸闷特点
- [ ] 胸痛/胸闷是从什么时候开始的？
- [ ] 是突然发作的，还是慢慢出现的？
- [ ] 疼痛的具体位置在哪里？（可以用手指指一下）
- [ ] 疼痛会放射到其他地方吗？（左肩、左臂、下颌、后背、牙齿）
- [ ] 疼痛是什么样的感觉？（压榨样、闷痛、针刺样、烧灼样、刀割样）
- [ ] 疼痛的程度如何？（1-10分，10分最痛）
- [ ] 每次疼痛持续多久？（几秒、几分钟、几十分钟、几小时）
- [ ] 疼痛发作的频率如何？（偶尔、每天几次、持续存在）

#### 2. 诱发与缓解因素
- [ ] 什么情况下会诱发？（活动、情绪激动、饱餐、寒冷、吸烟、平躺）
- [ ] 什么情况下会缓解？（休息、含服硝酸甘油、坐起）
- [ ] 硝酸甘油有效吗？几分钟起效？

#### 3. 心悸/心慌
- [ ] 有没有心悸、心慌的感觉？
- [ ] 心跳是太快了，还是太慢了，还是跳得不规律？
- [ ] 心悸是阵发性的还是持续性的？
- [ ] 发作时有没有摸过脉搏？跳得齐不齐？
- [ ] 发作时有诱因吗？（活动、情绪、咖啡、浓茶）

#### 4. 呼吸困难
- [ ] 有呼吸困难吗？感觉气不够用？
- [ ] 是活动后才出现，还是休息时也有？
- [ ] 能上几层楼？以前能上几层？
- [ ] 平躺时会加重吗？需要垫高枕头才能睡？
- [ ] 有没有睡着后突然憋醒，需要坐起来呼吸？

#### 5. 水肿
- [ ] 有没有水肿？哪里肿？（双下肢、眼睑、全身）
- [ ] 水肿是早上明显还是晚上明显？
- [ ] 按压后有凹陷吗？多久恢复？
- [ ] 最近体重有没有增加？

### 第二部分：伴随症状

#### 1. 全身症状
- [ ] 有没有头晕、头痛、黑蒙、晕厥？
- [ ] 有没有乏力、容易疲劳？
- [ ] 有没有出汗？特别是冷汗？
- [ ] 有没有发热？
- [ ] 最近体重有没有变化？

#### 2. 胃肠道症状
- [ ] 有没有恶心、呕吐？
- [ ] 有没有上腹胀痛？
- [ ] 有没有食欲不振？

#### 3. 呼吸道症状
- [ ] 有没有咳嗽？干咳还是有痰？
- [ ] 有没有咳痰？痰是什么颜色？有没有粉红色泡沫痰？
- [ ] 有没有咯血？

### 第三部分：病史相关

#### 1. 既往史
- [ ] 以前有没有确诊过心脏病？（冠心病、高血压、心律失常、心衰等）
- [ ] 有没有做过心电图、心脏彩超、冠脉造影等检查？
- [ ] 有没有高血压？血压一般多少？发现多久了？
- [ ] 有没有糖尿病？发现多久了？血糖控制如何？
- [ ] 有没有高血脂？
- [ ] 有没有脑血管病？（中风、脑梗塞）
- [ ] 有没有肾脏疾病？
- [ ] 以前有没有做过心脏手术？（支架、搭桥、起搏器等）

#### 2. 用药史
- [ ] 这次发病后有没有用过什么药？效果如何？
- [ ] 平时有没有长期服用的药物？（降压药、降糖药、降脂药、阿司匹林等）
- [ ] 最近有没有调整用药？
- [ ] 有没有用硝酸甘油？用了多久？

#### 3. 过敏史
- [ ] 有没有对什么药物过敏？
- [ ] 有没有对什么食物过敏？

#### 4. 个人史
- [ ] 吸烟吗？每天多少支？吸了多少年？
- [ ] 有没有戒烟？戒了多久？
- [ ] 喝酒吗？喝什么酒？大概多少量？喝了多少年？
- [ ] 平时饮食习惯如何？（是否偏咸、偏油）
- [ ] 平时运动吗？运动量如何？
- [ ] 工作压力大吗？睡眠怎么样？
- [ ] 体型偏胖还是偏瘦？

#### 5. 家族史
- [ ] 家人有没有心脏病？（冠心病、高血压、心律失常等）
- [ ] 有没有早发心脏病家族史？（男性<55岁，女性<65岁）
- [ ] 有没有糖尿病、高血压、高血脂家族史？
- [ ] 有没有猝死家族史？

## 鉴别诊断思考

### 根据症状鉴别

| 主要症状 | 可能疾病 | 鉴别要点 |
|----------|----------|----------|
| 胸痛 | 冠心病、心绞痛、心肌梗死、心包炎、胸膜炎、肺栓塞、气胸、胃食管反流 | 疼痛特点、诱发因素、持续时间、伴随症状 |
| 心悸 | 心律失常、窦性心动过速、早搏、房颤、甲状腺功能亢进 | 发作特点、持续时间、伴随症状 |
| 呼吸困难 | 心力衰竭、冠心病、心律失常、肺部疾病、贫血 | 发作特点、与体位关系、伴随症状 |
| 水肿 | 心力衰竭、肾脏疾病、肝脏疾病、下肢静脉疾病 | 水肿部位、出现时间、伴随症状 |
| 晕厥 | 心律失常、严重冠心病、体位性低血压、脑血管病、低血糖 | 发作特点、前驱症状、恢复情况 |

## 红色警报症状

出现以下情况需要立即建议就医（拨打120）：

- [ ] 持续剧烈胸痛超过20分钟不缓解
- [ ] 胸痛伴大汗、濒死感
- [ ] 胸痛伴呼吸困难、端坐呼吸
- [ ] 胸痛伴恶心呕吐、上腹痛
- [ ] 突然晕厥、意识丧失
- [ ] 严重呼吸困难、口唇发绀
- [ ] 咳出粉红色泡沫痰

## 问诊示例对话

```
用户: 我最近胸口闷，还有点痛

医生:
1. 这种胸闷胸痛是从什么时候开始的？是突然出现的，还是慢慢有的？
2. 具体是哪个位置不舒服？是胸口正中间，还是左边、右边？
3. 这种不舒服是什么样的感觉？是像石头压着一样闷痛，还是针刺样的痛？
4. 这种不舒服会持续多久？是几秒钟、几分钟，还是一直痛？
5. 什么情况下会诱发？比如走路快了、上楼时、情绪激动时？
6. 休息后会缓解吗？一般休息多久能好？
7. 除了胸痛，还有没有其他不舒服？比如心慌、出汗、恶心、呕吐？
8. 有没有头晕、眼前发黑？
9. 以前有没有高血压、糖尿病、高血脂？
10. 吸烟吗？每天多少？吸了多少年？
11. 家人有没有心脏病？
12. 这次有没有吃过什么药？比如硝酸甘油？有效吗？
```
